这一章，我们来聊一个非常具体、但极其重要的话题：出差。

我们组的研究理念是“面向国家实际需求，做好落地应用研究”。这意味着，走出实验室，去和合作方进行深入交流、部署测试、汇报成果，是我们的工作常态。出差，不是一场说走就走的旅行，而是你作为一名职业科研人员，在真实战场上的“客场作战”。

这篇文章的目的，就是把我们在过去无数次出差中用“血泪教训”换来的经验，总结成一份我们自己的作战清单。希望它能成为一个实用的标准流程，帮助我们所有人，尤其是刚开始接触这些的同学们，避开那些我们曾踩过的坑，让每一次外出都更从容、更高效。

## 一、 行前准备

出差的成功，一半取决于出发前的准备。

### **1.1【最重要】身份与权限：没有寸步难行**

- **身份证**：这是底线中的底线。没带身份证，虽然可以用 12306 上的临时身份证乘坐高铁、用支付宝->电子身份证入住酒店，但是我们的合作单位是肯定进不去的。
  - 我的习惯： 把它日常就放在手机壳里，形成肌肉记忆。
  - 万一真忘了：立即联系同组伙伴，用最快的快递寄到出差住宿的酒店。
- **多带至少一个其它可以代表身份的任意证件：** 比如社保卡、银行卡等，方便部分场所扣押。例如某些合作单位的保障酒店入住需要扣押一个证件，如果只有身份证，那就没法进所了。
- **通行报备**：**至少提前24小时，最好是提前两个工作日**，主动联系对方单位的对接人，提供你的姓名、身份证号、单位、联系方式等信息，完成访客报备。我们的合作方大多是安保严格的单位，没有报备，连大门都进不去，而且我们对接的合作伙伴都很忙，不一定会记得这事，需要我们自己格外上心。
- **天气与气候：** 我国幅员辽阔，南北气候差异巨大。提前查目的地天气和气温，带上雨伞和相应衣物，应对好湿冷或闷热，避免受冻感冒发烧。照顾好自己的身体，保持最佳状态。

### 1.2 【核心】工作准备：带着“武器”上战场

- **汇报材料：** 除了笔记本电脑，**不敏感**的 PPT、演示视频、技术文档等，务必再准备两个备份。 
  - **U盘备份：** 最稳妥的物理备份。
  - **云端备份**：发一份到自己网盘、邮箱，或存在微信文件里。
  - **PDF格式备份**：除了 PPTX 原文件，一定再导出一份 PDF。对方电脑可能没有你用的字体，或者Office版本不兼容，导致排版错乱，PDF能保证你的呈现效果始终如一。
- **设备与接口**： 
  - **笔记本电脑和充电器**：不多说。
  - **拓展坞（Docking Station）**：这是**重中之重**。你永远不知道对方会议室的投影仪是HDMI、VGA还是Type-C接口。为了避免“英雄无用武之地”的尴尬，必须带上拓展坞。**我办公室的柜子里有公用的，出差前随时来取。**
  - **移动光驱/光盘/移动硬盘/大容量U盘：** 以防万一我们要留些东西给合作方。
  - **随身 WIFI：** 网速更快、不担心流量，我和旻瑞这都有公用的，出差前随时来取。
  - **TF读卡器、SSD硬盘盒子、HDMI转DP接口、小型屏幕：** 方便板卡部署临时需求，无此要求的可以无视。
- **【痛点】全离线环境的生存法则**
  - 必须清醒地认识到，我们去的很多合作单位，实验室是 **物理隔绝（Air-Gapped）** 的，没有外网，甚至连手机信号都被屏蔽。千万不要抱有“到了现场再配环境”、“缺了包再下载”的幻想。在出发前，请务必做好“断网生存”的准备。
  - **Docker 镜像与离线包：** 如果涉及代码部署，请务必打包好完整的 Docker 镜像，或者下载好所有依赖库的 .whl 文件。不要只带代码，要带环境。
  - **本地文档：** 把可能会用到的 API 文档、技术手册下载成 PDF 或 HTML 离线版。到了现场，Stack Overflow 和 CSDN 是不存在的。 
  - **自带算力：** 如果条件允许，尽量在我们带 GPU 的笔记本上跑通一个 Demo。不要过度依赖对方提供的服务器，对方的 CUDA 版本或者驱动可能根本不兼容你的代码框架。
- **【红线】安全与合规**
  - **电子设备合规：** 前往有明确要求的合作单位（尤其是国家重点单位），所有随身带入合作单位的电子设备，包括笔记本电脑、手机、降噪耳机、电子手表等，只能是**国产品牌**。不要给我们的合作伙伴带来不必要的麻烦。
  - **安全意识：** 在合作单位内部，不随意拍照，绝对不手贱或者脑残将私人存储设备插入对方电脑。
- **【个人】形象与健康**
  - **个人护理：** 男生别忘了剃须刀（这个放在浴室最容易忘）。如果实在忘了，大部分酒店前台提供一次性剃须刀和泡沫。注意：气压罐装的剃须泡沫不能带上飞机高铁，我通常用洗面奶、洗手液临时替代，或者直接用酒店提供的泡沫。
  - **健康防护：** 口罩、一次性马桶垫、消毒湿巾。特别是在流感高发季或长途出行时，这些小东西能极大提升舒适度和健康保障。我办公室柜子里常备这些“出行神器”，欢迎大家自取。
  - **常用药品：** 根据个人情况，带上肠胃药、感冒药、创可贴等。

## 二、 差旅中：把控风险，高效利用时间

### 2.1 交通与时间：将不确定性降至最低

- **北京出行策略：** 在其他城市，打车或许更高效舒适。但**在北京，地铁优先是铁律**。我们的标准做法是：乘坐地铁到达离目的地最近的地铁站，再短距离打车。直接打车纵穿市区，拥堵的风险极高，很可能让我们错过至关重要的时间点。
- **预留充足时间：** 凡事赶早不赶晚。无论是赶火车飞机，还是去对方单位，都要比你预估的时间再早30分钟。

### 2.2 财务与报销：让流程顺畅，不为琐事分心

- **遵守标准：** 按照学校的财务规定标准选择交通工具和住宿酒店，从源头避免后续报销的麻烦。
- **住宿单据：** 如果多日出差的住宿费超过 1000元，按学校要求，务必在退房时向前台索要并保管好住宿清单（水单），或者确认发票备注栏已清晰体现住宿人、入住离店时间等信息。这是报销的必需文件，提前做好，以免事后补救困难。

## 三、 结语 

出差，绝不仅仅是换个地方写代码。 它是对大家项目管理能力、沟通协作能力、风险控制能力以及职业素养的一次综合演练。 在这个过程中，我们与合作方建立起的面对面的信任与友谊，我们对国家实际需求的切身感悟，其价值远超一篇论文。 

希望大家每次出发前，都能对照这份清单 Check 一遍。当你能从容不迫、游刃有余地处理好这一切琐碎时，你就离一名成熟、独立的“准 PI”又近了一步。
